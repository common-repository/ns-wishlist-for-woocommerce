== Description ==

= WHAT IT DOES =
This plugin allows your visitors and registered users to create custom wishlist.
Creating own wishlist, users can distribute copies of this to family and friends. 
Sharing the wishlist the recipient is able to purchase these gifts. 
The goal of a wish list is to facilitate communication between the gift receiver and the gift giver and, 
furthermore, increase sales and conversions for you.
Display wishlist is very easy thanks to shortcode! No code required!

= HOW IT DOES IT =
When non registered users use NS Wishlist For Woocommerce a new cookie is created and managed. 
This means that cookie after some time will deleted. 
Otherwise if user is registered, the wishlist will saved for ever in your database.
Wishlist saved by cookie will converted and associato to the user account during log in.

== Installation ==

1. Upload the entire 'ns-woocommerce-wishlist' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on 'Wishlist' and start to customize it.
